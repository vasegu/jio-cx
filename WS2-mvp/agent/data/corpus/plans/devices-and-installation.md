# Devices and Installation - JioFiber & JioAirFiber

## JioFiber Router

### Router Models
Jio has provided different router models over time. The current standard-issue router:

- **Jio Home Gateway** (dual-band Wi-Fi router)
  - Wi-Fi 5 (802.11ac) on older installations, Wi-Fi 6 (802.11ax) on newer ones
  - Dual-band: 2.4 GHz + 5 GHz
  - 2-4 Gigabit Ethernet LAN ports
  - 1 WAN port (fiber input via ONT)
  - USB port (select models)
  - Supports WPA2/WPA3 security
  - Managed by Jio - firmware updates pushed remotely

### ONT (Optical Network Terminal)
- A small box that converts the fiber optic signal to Ethernet
- Installed by the Jio technician, usually mounted near the fiber entry point
- Connected to the router via Ethernet cable
- Powered separately (needs its own power outlet)
- The ONT is Jio's property and must be returned if service is cancelled

### Jio Set Top Box (Optional)
- **Price:** INR 1,999 one-time purchase (approximate)
- Connects to TV via HDMI
- Runs Jio's interface with all OTT apps pre-installed
- Voice-enabled remote control
- Supports 4K output
- Can be used as a standalone streaming device even without JioFiber (via any internet)
- Not required for broadband service - only needed if you want OTT apps on your TV without a smart TV

## JioAirFiber Device

### Indoor CPE (Customer Premises Equipment)
- **Form factor:** Compact cylindrical or rectangular unit (varies by generation)
- **Technology:** 5G FWA (Fixed Wireless Access) gateway
- **Built-in Wi-Fi:** Wi-Fi 6 (802.11ax), dual-band 2.4 GHz + 5 GHz
- **Ethernet ports:** 1-2 Gigabit Ethernet LAN ports for wired connections
- **HDMI output:** Yes - connects directly to TV for streaming content
- **Power:** Standard wall outlet (power adapter included)
- **Voice remote:** Included for navigating the built-in media interface
- **Dimensions:** Compact enough to sit on a shelf or table

### Key Differences from JioFiber Router
- No ONT needed (signal is received wirelessly from 5G tower)
- Built-in media streaming capabilities (acts as a set-top box)
- HDMI output for direct TV connection
- Must be positioned where 5G signal is strong (preferably near a window)
- Single device replaces router + ONT + set-top box

## Installation Process

### JioFiber Installation

**Step 1: Registration and Eligibility Check**
- Customer registers on jio.com or MyJio app
- Enters address to check fiber availability
- Selects preferred plan
- Schedules installation appointment

**Step 2: Technician Visit (Day 1 or within 1-3 days)**
- Jio-authorized technician arrives at scheduled time
- Surveys the premises for fiber cable routing
- Lays fiber optic cable from the nearest distribution point to the home
- Cable may be routed along walls, through conduit, or overhead depending on building type
- In apartments/societies with pre-existing Jio infrastructure, this is faster

**Step 3: Equipment Installation**
- Installs the ONT (Optical Network Terminal) near the fiber entry point
- Connects the Jio router to the ONT via Ethernet
- Powers on both devices
- Configures the Wi-Fi network (SSID and password)

**Step 4: Activation and Testing**
- Technician activates the connection from their app
- Runs speed test to verify the plan speed is being delivered
- Ensures Wi-Fi is working throughout key areas of the home
- Demonstrates basic features to the customer

**Step 5: Customer Verification**
- Customer verifies internet connectivity on their devices
- Confirms the plan and billing details
- OTT apps are activated (may take a few hours to fully reflect)

**Timeline:** Typically 1-3 days from registration to active connection. In areas with existing Jio infrastructure (pre-wired societies), same-day installation is possible.

### JioAirFiber Installation

**Step 1: Registration and Eligibility Check**
- Customer registers on jio.com or MyJio app
- Enters address to check 5G coverage availability
- Selects preferred plan
- Schedules installation appointment

**Step 2: Technician Visit (usually same day or next day)**
- Technician arrives with the JioAirFiber device
- Checks 5G signal strength at the premises
- Identifies the optimal placement for the device (usually near a window facing the 5G tower)

**Step 3: Device Setup**
- Places the JioAirFiber device in the optimal location
- Connects power cable
- If customer wants TV streaming: connects HDMI cable to TV
- Powers on the device

**Step 4: Activation and Testing**
- Technician activates the device from their app
- Verifies 5G signal lock and internet connectivity
- Runs speed test
- Configures Wi-Fi name and password

**Step 5: Customer Handover**
- Demonstrates Wi-Fi connectivity
- Shows how to use the voice remote for TV content (if HDMI connected)
- Confirms plan and billing

**Timeline:** Usually same day or next business day. Installation takes 30-60 minutes since no cabling is required.

## What the Customer Needs to Provide

### For JioFiber
- **Power outlet** near the desired router location (for both ONT and router)
- **Valid ID proof** (Aadhaar, PAN, passport, etc.) for KYC verification
- **Address proof** if ID address differs from installation address
- **Access permission** - if in an apartment, society permission may be needed for cable routing
- **Clear path** for fiber cable routing from building entry to the flat/house
- **Payment method** - for first month's plan payment and any applicable security deposit

### For JioAirFiber
- **Power outlet** near a window or area with good 5G signal
- **Valid ID proof** for KYC verification
- **HDMI-compatible TV** (optional, only if using the streaming feature)
- **Payment method** for first month's plan payment

## Troubleshooting and Support

- **MyJio App:** Manage plan, check usage, raise complaints, chat with support
- **Jio Customer Care:** Call 198 (from Jio number) or 1800-889-9999 (toll-free)
- **Jio.com:** Online chat support, FAQ, service request tracking
- **Technician Revisit:** Can be requested via MyJio app if hardware issues arise
- **Self-service:** Router can be rebooted by customer; most connectivity issues resolve with a power cycle of both ONT and router

## Equipment Return Policy

- Router, ONT, and AirFiber device are Jio's property
- Must be returned in working condition upon service cancellation
- Security deposit is refunded after equipment return and inspection
- Damaged or unreturned equipment may result in forfeiture of security deposit
