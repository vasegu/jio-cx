# JioAirFiber Plans

JioAirFiber is Jio's Fixed Wireless Access (FWA) broadband service. It delivers high-speed internet wirelessly via Jio's 5G network, requiring no fiber cable to the home. Ideal for areas where JioFiber has not yet been deployed.

## How JioAirFiber Works (Fixed Wireless Access)

1. **No cable needed** - Internet is delivered over Jio's 5G radio network, not a physical fiber line
2. **Indoor CPE device** - A compact indoor gateway device (the JioAirFiber unit) receives the 5G signal and creates a Wi-Fi network in the home
3. **Plug and play** - Technician installs the device, positions it for optimal signal, and activates the connection
4. **Coverage dependent** - Performance depends on Jio 5G coverage strength at the customer's location
5. **No digging or drilling** - Since there is no cable, installation is non-invasive and fast

## JioAirFiber Device Details

### Indoor Gateway (CPE)
- **Form factor:** Compact indoor unit, roughly the size of a small speaker
- **Connectivity:** Built-in Wi-Fi 6 router (dual-band 2.4 GHz + 5 GHz)
- **Ports:** Ethernet LAN ports (for wired connections), power input
- **HDMI output:** Some models include HDMI out for direct TV connection (acts as a set-top box)
- **Setup:** Powered via standard wall outlet, no special wiring
- **Positioning:** Should be placed near a window or in an area with strong 5G signal reception

### Built-in Smart TV Features
- The JioAirFiber device doubles as a media streaming box
- HDMI output connects directly to any TV
- Access to JioTV+, JioCinema, and all included OTT apps via the device
- Voice remote control included for navigating content

## JioAirFiber vs JioFiber Comparison

| Feature | JioFiber | JioAirFiber |
|---------|----------|-------------|
| Technology | FTTH (Fiber to the Home) | FWA (Fixed Wireless Access / 5G) |
| Cable required | Yes (fiber optic) | No |
| Speed consistency | Very consistent (wired) | Depends on 5G signal strength |
| Latency | Very low (~2-5ms) | Low (~5-15ms, varies) |
| Installation time | 1-3 days (fiber laying may be needed) | Same day (plug and play) |
| Coverage | 150+ cities, locality-dependent | Wherever Jio 5G is available |
| Plans and pricing | Same tiers | Same tiers and pricing |
| OTT bundles | Same | Same |
| Built-in STB | Separate Jio STB (optional) | Built into AirFiber device |

## Availability

JioAirFiber requires Jio 5G coverage at the customer's location. Available in cities and towns with Jio's 5G network. Customers can check eligibility on jio.com or the MyJio app by entering their address.
